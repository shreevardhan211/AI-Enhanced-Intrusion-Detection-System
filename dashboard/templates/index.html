🛡️ Project Overview

The AI-Enhanced Intrusion Detection System is a real-time network security monitoring platform using machine learning to detect and classify malicious traffic.

Built on the UNSW-NB15 dataset, it trains a Random Forest classifier to distinguish normal traffic from attack categories: DoS, Reconnaissance, Exploits, Fuzzers, and more.

The SOC Dashboard gives a full view of network health, live packet analysis, attack trends, and model metrics.
